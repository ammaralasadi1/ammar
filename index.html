Explore & Redux

A single page application that combines weather, maps, music and news. It offers exploration by going random to places around the world (feeling lucky button) or choosing your own destination.

created with: Redux library - React - Axios - JSON - express.Js - Weather, Google maps, Music and news APIs

DC Cars and Cofee

Automotive online magazine and marketplace. DC Cars and Coffee is a full stack Ruby on Rails application. You can share articles or list cars for sale within the community. It features categories, commenting and full user authentication to protect users content. Full CRUD on 3 models and nested routes.

created with: Ruby on Rails - Devise gem - PostgreSQL - HTML5 - CSS3

Weatro

WeaTro is a single page application to get metro arrival times and weather for a selected WMATA station. This application was a group project to practice agile methodologies and Git, GitHub workflow. It features Angular.Js front end and Ruby on Rails backend. Users can also rate the stations by leaving comments.

created with: Angular.Js - Ruby on Rails - AJAX - JSON - WMATA, Underground Weather APIs - jQuery - HTML5 - CSS3

Math Trivia

Math Triva is quiz game built with the user in mind. It features simple game logic and animations. The user can answer questions, get hints and keep score while playing. The application saves the scores in the browser using localStorage.

created with: javaScript - jQuery - HTMl5 - CSS3
